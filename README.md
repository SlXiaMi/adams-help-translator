# Adams Help Translator

一键配置工具，让 Adams 2024.1 帮助文件支持 Edge 浏览器翻译。

## 原理

Adams 按 F1 默认打开 `file://` 协议的本地帮助文件，Edge 翻译功能只对 `http://` 页面生效。本工具通过两个步骤解决：

1. **注入跳转脚本** — 在所有帮助文件中注入自动跳转代码，将 `file://` 页面重定向到本地 HTTP 服务
2. **后台 HTTP 服务** — Node.js 静态文件服务器，开机自启，无窗口静默运行

## 使用方法

### 配置（仅一次）

右键 `setup.bat` → **以管理员身份运行**

### 日常使用

在 Adams 中按 F1 → 自动跳转 → 点击 Edge 地址栏翻译图标 → 中文翻译

### 恢复

右键 `recover.bat` → **以管理员身份运行**

## 文件说明

| 文件 | 用途 |
|------|------|
| `server.js` | HTTP 静态文件服务（端口 8777） |
| `inject.js` | 批量注入跳转脚本到帮助文件 |
| `recover.js` | 批量移除注入，恢复原状 |
| `setup.bat` | 一键配置脚本 |
| `recover.bat` | 一键恢复脚本 |

## 依赖

- Node.js（已随 Adams 2024.1 安装）
- Edge 浏览器
